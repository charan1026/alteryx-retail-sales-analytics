# Retail Sales Analytics Pipeline (Alteryx + Power BI)

This project demonstrates a retail sales analytics workflow built using Alteryx for data preparation and Power BI for visualization.

## Tools Used
- Alteryx Designer
- Power BI
- CSV Dataset

## Workflow Steps
1. Data cleaning using Alteryx
2. Data type conversion and filtering
3. Aggregation of sales data by region, category, and sub-category
4. Identification of top-performing products
5. Visualization of insights using Power BI

## Key Insights
- Technology category generated the highest sales.
- Phones and Chairs were among the top-selling sub-categories.
- The West region produced the highest overall sales.

## Dashboard
The Power BI dashboard highlights:

- Sales by Region
- Sales by Category
- Top 5 Sub-Categories
- Total Sales, Profit, and Quantity KPIs
